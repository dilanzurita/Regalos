Animaciones

JavaScript HTML DOM Animación

¿Qué es?

Son efectos basados en movimientos o transiciones generadas por scripts para hacer más atractiva nuestra página

Como funciona

Se llevan a cabo mediante la programación de los cambios graduales en el estilo de un elemento. Los cambios se llaman por un temporizador. Cuando el temporizador de intervalo es pequeño, la animación se ve continua.

Uso

Su uso principal es para dar un toque estetico e interactivo a la página

Eventos

JavaScript HTML DOM Eventos

¿Qué es?

Es una acción que un JavaScript puede ejecutar cuando se produce un evento, como cuando un usuario hace clic en un elemento HTML.

Importancia

Los eventos son importantes para indicarle al usuario el inicio, transicion o fin de una acción, de esta manera se podrá evitar confuciones y habrá mejor precision en los procesos.

Ejemplos de uso

- Cuando un usuario hace clic en el ratón
- Cuando una página web se ha cargado
- Cuando se cambia un campo de entrada
- Cuando se envía un formulario HTML

Eventlistener

JavaScript HTML DOM EventListener

¿Que es?

El método addEventlistener, es un escuchador que indica al navegador que este atento a la interacción del usuario. La ventaja es que se escribe totalmente en el JS, sin necesidad de tocar el HTML.

Metodos más usados

- El addEventListener() método se conecta un controlador de eventos para el elemento especificado.
- El removeEventListener() método elimina los controladores de eventos que se han adjuntado a la addEventListener() método

Uso

- Usted puede agregar muchos controladores de eventos a un elemento.
- Se pueden añadir detectores de eventos a cualquier objeto DOM no sólo los elementos HTML. es decir, el objeto de la ventana.

Node

JavaScript Elementos DOM de HTML (nodos)

¿Qué es?

Un nodo es un único punto en el arbol de nodos. Los nodos pueden ser varias cosas el documento mismo, elementos, texto y comentarios.

Funciones

appendChild()
insertBefore()
removeChild()

Uso

- Añadir un elemento
- Eliminar un elemento
- Sustituir un elemento
- Crear un elemento

Colecciones

JavaScript HTML DOM - Colecciones

¿Que són?

Objetos ordenados de una forma específica la cual se puede manipular segun se requiera

Características

- Casi todas las colecciones del DOM, salvo algunas excepciones, están vivas. En otras palabras, reflejan el estado actual del DOM.
- Las colecciones son iterables usando for..of. Algunas veces las personas tratan de utilizar for..in para eso.
- Las colecciones tienen unas propiedades “extra” raramente usadas que normalmente no queremos obtener
- Algunos tipos de elementos del DOM, por ejemplo las tablas, proveen propiedades adicionales y colecciones para acceder a su contenido.

Uso

- Ir a sus inmediatos vecinos utilizando las propiedades de navegación.
- Añadimos o quitamos nodos del DOM, entonces estos nodos aparecen en la colección automáticamente.
- No podemos reemplazar a un hijo por otro elemento asignándolo así childNodes[i]

Lista de Nodos

JavaScript Lista HTML DOM nodo

¿Que són?

Una lista de nodos es una colección de tipo vector de nodos.

Funciones

- Seleccion de una etiqueta en todo el documento
- Determinar la longitud de una lista
- Modificacion de las propiedades de una etiqueta

Uso

Facilita la busqueda, organizacion y modificacion de los nodos (etiquetas) que estamos usando en nuestra web.

Navegacion

JavaScript Navegación DOM HTML

Navegacion

Con el DOM de HTML, se puede navegar por el árbol de nodos utilizando relaciones de nodo.

Propiedades

- parentNode
- childNodes[ nodenumber ]
- firstChild / lastChild
- nextSibling / previousSibling

Uso

- Obtener contenido de un elemento
- Acceso al documento completo
- Especificar características de un nodo
